Responsive Landing Page Prototype

This project is a fully responsive landing page prototype built with React and styled with Tailwind CSS. It was designed to showcase a modern product-focused interface with clean layout structure and strong visual consistency across different screen sizes. The application includes responsive navigation, feature highlights, workflow and pricing sections, testimonials, and a structured footer. I also integrated Lucide React icons to enhance the UI and improve visual clarity. The project was deployed on Vercel, and I applied WCAG-aligned accessibility improvements such as semantic HTML, keyboard-friendly navigation, visible focus states, and accessible labels.

Fast React Pizza

This project was built from scratch as a practice activity for Jonas's React course. It leverages React Router and Redux Toolkit for state management, as well as the useFetcher hook to retrieve data directly from routes. Tailwind CSS was used for styling and responsive layouts. Throughout the project, I gained hands-on experience building reusable components, such as the custom Button.jsx. Additionally, I went beyond the tutorial by applying WCAG accessibility principles to improve usability and inclusivity. Finally, it was deployed on Netlify.

Info Music

A responsive React application that surfaces essential info about bands and artists. Type a name to get up to three results with image carousels, country, discography, and a concise biography. A Node/Express backend aggregates data from MusicBrainz, Cover Art Archive, Last.fm, Deezer, and Wikipedia. Debounced search, spinners, and placeholders keep the UX smooth.

React Quiz

React quiz app with 100 questions. Originally a capstone project from Jonas Schmedtmann's React course, I extended it by adding features like question count selection, elapsed time tracking, question shuffling, and high score logic. I also enhanced the UI, made it fully responsive, and optimized it for SEO and social sharing.

Movie Web App

This project was created as an exercise in Jonas Schmedtmann's React course. I practiced concepts like useEffect to fetch data from the OMDB API, useState, useRef, and prop drilling. The app lets users search for movies, view details, and rate them. I also personally made the interface fully responsive, which helped me strengthen my skills with React hooks, API integration, and responsive design.

Eat and Split Web App

Also built with React, this app is designed to make splitting bills with friends easy and stress-free by automatically handling all the calculations for you.
